# カテゴリ移行ガイド

## 変更内容

### 光熱費カテゴリの削除と固定費への統合

**変更前**:
```
- 光熱費
  - 電気代
  - ガス代
  - 水道代
- 固定費
  - 家賃
  - 投資
  - 通信費
```

**変更後**:
```
- 固定費
  - 家賃
  - 投資
  - 通信費
  - 電気代  ← 移動
  - ガス代  ← 移動
  - 水道代  ← 移動
```

## 既存データへの影響

### 自動的に対応される項目
- **新規取引**: 新しいカテゴリ構造で入力可能
- **カテゴリ選択**: 「光熱費」が選択肢から削除される

### 手動対応が必要な項目
- **既存の取引データ**: 過去に「光熱費」カテゴリで登録された取引は、データベース上ではそのまま残ります
- **既存のテンプレート**: 「光熱費」カテゴリのテンプレートは、データベース上ではそのまま残ります

## 既存データの扱い

### オプション1: そのまま使用（推奨）
既存の「光熱費」データはそのまま残しておく：
- **メリット**: 過去のデータが保持される、履歴が正確
- **デメリット**: 統計に「光熱費」と「固定費」が別々に表示される可能性

### オプション2: 手動で編集
必要に応じて、過去の取引を手動で編集：
1. 取引一覧から「光熱費」の取引を探す
2. 編集ボタンをクリック
3. カテゴリを「固定費」に変更
4. サブカテゴリを適切に選択（電気代/ガス代/水道代）
5. 保存

### オプション3: Firebaseコンソールで一括変更（上級者向け）
Firebase Consoleから直接データを編集：

1. **Firebase Console にアクセス**
   - https://console.firebase.google.com
   - プロジェクトを選択

2. **Firestore Database を開く**
   - 左メニューから「Firestore Database」を選択

3. **transactions コレクションを開く**
   - `users/{userId}/transactions` を探す

4. **光熱費の取引を検索**
   - フィルタ機能を使用して `category == "光熱費"` で検索

5. **各ドキュメントを編集**
   - `category` フィールドを `"固定費"` に変更
   - `subcategory` フィールドはそのまま（電気代/ガス代/水道代）

## テンプレートの扱い

### 既存の光熱費テンプレート
「光熱費」カテゴリのテンプレートは使用できなくなります：
- テンプレート一覧には表示されますが、使用時にエラーになる可能性があります
- 新しく「固定費」カテゴリでテンプレートを作成し直すことを推奨

### 対応方法
1. テンプレート一覧を開く
2. 「光熱費」のテンプレートを削除
3. 新しく「固定費」カテゴリでテンプレートを作成

## 統計・グラフへの影響

### 円グラフ
- 新規データ: 「固定費」として集計される
- 既存データ: 「光熱費」として集計される（データを移行しない場合）

### 月次比較
- 過去の「光熱費」と現在の「固定費」は別カテゴリとして扱われる
- 正確な比較のためには、過去データの移行を推奨

## 推奨される対応手順

### 新規ユーザー
- 特に対応不要
- 新しいカテゴリ構造で使用開始

### 既存ユーザー（少量のデータ）
1. 新しいカテゴリ構造で取引を入力開始
2. 必要に応じて過去の重要な取引を手動で編集
3. 古いテンプレートを削除し、新しく作成

### 既存ユーザー（大量のデータ）
1. 新しいカテゴリ構造で取引を入力開始
2. 過去データはそのまま保持（履歴として）
3. 統計を見る際は「光熱費」と「固定費」を合算して考える
4. または、Firebaseコンソールで一括変更（上級者向け）

## よくある質問

### Q: 過去のデータは自動的に移行されますか？
A: いいえ、自動的には移行されません。必要に応じて手動で編集してください。

### Q: 光熱費のデータを残しておいても問題ありませんか？
A: はい、問題ありません。過去のデータとして保持されます。

### Q: 統計に影響はありますか？
A: 「光熱費」と「固定費」が別カテゴリとして集計されます。正確な統計のためには、データの移行を推奨します。

### Q: テンプレートはどうなりますか？
A: 既存の「光熱費」テンプレートは使用できなくなる可能性があります。新しく作成し直すことを推奨します。

## まとめ

カテゴリ構造の変更により、光熱費が固定費に統合されました。新規データは新しい構造で入力できますが、既存データは必要に応じて手動で移行してください。
